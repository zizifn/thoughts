# User

## Command

1. useradd
    `useradd wilson`
2. id
    `id root`
3. userdel
    `userdel wilson` or `userdel -r wilson `
4. usermod, 修改用户
    ```
    uid=0(root) gid=0(root) groups=0(root)
    ```
5. groupadd
6. su sitch user
    `su - wilson`
7. sudo
    use `visudo`, to give normal user access to some command.
