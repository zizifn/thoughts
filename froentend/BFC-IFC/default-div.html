没有标签的文字，会被默认一个box..span 标签内的元素



------------------------------------------------------------------------------
  • - HTML 标签会创建一个BFC, 这说明浏览器上来就准备好了一切。
  • - 每个元素/box都有自己的命，是应该放在行中还是列？上天已经给了一个default 值
  • - div 上天给的default 值是display: block， F12可以查看。所以他被放入列中， 所以两个div会被叠罗汉。MDN doc
  • - 遇到需要放入行中的元素，如果没有IFC，会创建IFC。
  • - span 上天给的default 值是display:inline， 所以它不会被放到列中。
  • - 可不可以逆天改display， 可以，但是也许会有点不完美。 MDN doc


    • ------------也许你也发现，元素/box 里面也可以放入新的元素/box. 比如div1------------------------------------------------------------------
  • - 每个元素/box 里面可不可以放入列，换句话说就是，元素内部可不可以有BFC， 这也是上天定的。
  • - 比如div里面就可以把元素排成列，然而span 就不可以。
  • - 当然同样可以通过改display，来逆天改命。MDN doc
